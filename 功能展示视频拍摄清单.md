# 智能日语词汇记忆辅助系统功能展示视频拍摄清单

建议视频时长控制在 5-8 分钟。录屏时按下面顺序展示，基本可以覆盖课程老师最关心的“功能完整性、AI 方法、数据分析、实用性和界面效果”。

## 0. 开场准备

- 打开浏览器访问 `http://localhost:8501`。
- 简单展示系统名称和左侧功能目录。
- 说明这是一个本地运行的 Streamlit 日语学习系统。
- 展示当前词库下拉框和今日学习目标滑块。

## 1. 背单词页面

- 选择一个词库，例如 `MOJi N2汉字考前对策`。
- 调整“今日学习目标”滑块，例如调到 10 或 50。
- 展示顶部统计卡片：词汇总量、已掌握、待学习、今日建议复习。
- 展示推荐队列下拉框。
- 展示单词、假名、释义、词性、JLPT 等级、遗忘风险。
- 点击“发音”按钮。
- 点击“加入错题本”。
- 点击一次“认识 / 模糊 / 不认识”中的一个，展示统计数据会变化。
- 简单说明：用户反馈会写入数据库，并用于后续遗忘风险和学习分析。

## 2. 自适应自测页面

- 切换到“自测”页面。
- 展示“重新开始自测”按钮。
- 展示题干和四个选项，说明选项中混合日语汉字、假名和中文释义。
- 选择一个答案并点击“提交答案”。
- 如果答对，展示自动进入下一题；如果答错，展示正确答案会停留在当前题。
- 展示本轮进度、正确率。
- 如果已有 20 题以上记录，展示能力评估与学习建议。
- 打开“答题数据分析”，展示题型正确率和 JLPT 等级正确率。
- 说明：系统根据词汇难度加权评估 N5-N1 水平。

## 3. 高频词分析页面

- 切换到“高频词分析”页面。
- 展示 JLPT 等级筛选和高频词选择。
- 选择一个高频词，展示词义、读音、用法和例句。
- 在文本框输入一小段日语文本，例如：
  `今日は日本語の単語を勉強して、ニュースの記事を読みました。`
- 点击分析或等待结果展示。
- 展示词频表、N-gram 搭配分析和词云图。
- 说明：该模块体现日语文本分析、分词、词频统计和可视化。

## 4. 学习分析页面

- 切换到“学习分析”页面。
- 点击“刷新分析数据”。
- 展示“最近一次更新时间”。
- 展示学习概览指标：累计学习天数、总复习次数、总自测题数、连续学习天数。
- 展示每日学习趋势图。
- 展示 JLPT 等级掌握率图。
- 展示累计掌握词汇增长图。
- 展示错题词性分布图。
- 展示遗忘风险排行榜。
- 展示学习画像雷达图和系统建议。
- 说明：图表会随着背词和自测数据变化，不是静态图片。

## 5. 项目说明页面

- 切换到“项目说明”页面。
- 展示项目定位和当前数据统计。
- 展示“功能模块”表格。
- 展示“AI 与数据分析体现”表格。
- 展示“数据库设计”表格。
- 展示“大作业要求对应”表格。
- 说明：该页面方便老师快速理解系统如何满足课程要求。

## 6. 自检与数据质量页面

- 切换到“自检”页面。
- 展示当前词库是否存在字段缺失、假名格式异常、重复词条等问题。
- 展示用户反馈记录。
- 展示 Logistic Regression 和 Random Forest 的模型评价指标。
- 说明：自检用于保证词库质量，避免错误数据影响学习。

## 7. 词库管理页面

- 切换到“词库管理”页面。
- 展示当前词库表格。
- 展示“重命名已有词库”区域。
- 展示“导入 CSV / Excel”区域。
- 展示可以自定义导入后的词库名称。
- 展示删除词库区域。
- 说明：系统支持用户导入自己的教材词表，形成个性化学习系统。

## 8. 错题本页面

- 切换到“错题本”页面。
- 展示错题本数量。
- 选择一个错题词。
- 展示单词、假名、释义、例句、JLPT 等级和来源词库。
- 点击“发音”。
- 展示“移出错题本”按钮，但录屏时可以不真的删除。
- 展示下面的错题列表。
- 说明：错题本用于集中复盘薄弱词汇。

## 9. 文本转语音页面

- 切换到“文本转语音”页面。
- 输入一段日语，例如：
  `日本語の発音を練習します。`
- 选择一个日语声线。
- 调整语速。
- 点击生成语音。
- 展示音频播放器和 MP3 下载按钮。
- 说明：该功能用于发音模仿和听力辅助。

## 10. 中日互译页面

- 切换到“中日互译”页面。
- 展示翻译方向选择：自动识别、日语 → 中文、中文 → 日语。
- 输入日语句子：
  `今日はいい天気です。`
- 点击“翻译”，展示中文结果。
- 点击“清空结果”。
- 输入中文句子：
  `我每天复习日语单词。`
- 选择“中文 → 日语”并点击翻译。
- 展示日语结果。
- 说明：该功能不需要 API Key，适合查词、短句理解和阅读辅助；联网不可用时会给出错误提示。

## 11. 总结收尾

- 回到“学习分析”或“项目说明”页面。
- 总结系统亮点：
  - 机器学习记忆预测。
  - 遗忘风险智能推荐。
  - 自适应自测和能力评估。
  - 高频词与文本分析。
  - 学习数据可视化。
  - 自定义词库和 MOJi PDF 词库。
  - 错题本、TTS、中日互译等实用辅助功能。
  - 极简复古杂志风 UI。

## 视频拍摄注意事项

- 每个页面停留 15-40 秒，重点页面可以稍长。
- 不要只快速切页面，要至少演示一次真实交互。
- 自测答题时可以提前准备几个会答的题，避免录屏时卡住。
- TTS 和中日互译依赖网络，录制前先测试一次。
- 如果翻译接口临时失败，可以展示错误提示，并说明系统有联网状态提示。
- 截图和视频尽量使用同一套词库，显得报告和演示一致。
