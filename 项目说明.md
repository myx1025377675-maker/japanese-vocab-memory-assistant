# 智能日语词汇记忆辅助系统项目说明

## 一、项目目标

本项目面向日语学习者，设计一个本地运行的智能词汇学习系统。系统通过学习记录、机器学习预测和文本词频分析，辅助用户安排复习顺序、检测薄弱词汇，并支持自定义词库导入。

## 二、AI 与数据分析体现

1. 记忆效果预测：使用模拟学习行为数据训练 Logistic Regression 与 Random Forest，特征包括 JLPT 难度、复习次数、平均掌握度、最近掌握度、复习间隔等。
2. 智能推荐：根据模型输出的记忆保持概率计算遗忘风险，结合复习间隔和新词优先级生成推荐队列。
3. 自适应自测：依据用户历史答题正确率和词汇难度调整题目抽样权重。
4. 高频词分析：对用户输入的日语文本进行规则分词，统计词频并生成词云。
5. 数据自检：检查词库字段、假名格式、重复词条和长期低掌握词，支持用户反馈记录。

## 三、主要技术

- Python
- Streamlit
- SQLite
- Pandas / NumPy
- Scikit-learn
- Matplotlib / WordCloud

## 四、运行方式

```powershell
pip install -r requirements.txt
streamlit run app.py
```

Windows 环境也可以直接运行：

```powershell
.\run_app.bat
```

## 五、数据说明

首次运行会自动创建 `japanese_learning.db`，并写入示例 JLPT 核心词库。用户可以在“词库管理”页面导入 CSV 或 Excel，必要列为 `word/kana/meaning` 或 `单词/假名/释义`。

## 六、可改进方向

- 接入 Janome 或 MeCab，提高日语分词和读音校验准确率。
- 使用真实长期学习记录替代模拟训练数据。
- 增加更完整的听力题和例句填空题。
- 扩充 JLPT N5-N1 全量词库。
